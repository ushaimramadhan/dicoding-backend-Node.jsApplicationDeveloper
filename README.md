# Node.js Back-End Submission

Repositori ini berisi submission akhir kelas **Menjadi Node.js Application Developer**.

## Fitur & Kemampuan Utama

- **Stream & Buffer:** Mengolah aliran data besar secara efisien tanpa membebani memori.
- **Sistem File (fs):** Melakukan manipulasi berkas dan direktori menggunakan modul Filesystem.
- **Child Process:** Menjalankan proses latar belakang dan mengotomatiskan tugas sistem.
- **Unit Testing:** Pengujian kode secara otomatis untuk memastikan fungsionalitas aplikasi berjalan sesuai ekspektasi.

## Teknologi yang Digunakan

- **Language:** JavaScript
- **Environment:** Node.js
- **Testing Framework:** Node.js Test Runner / Jest

## Struktur Folder

- `Pertama/`: Implementasi dasar modul Node.js (Buffer & Streams).
- `Kedua/`: Implementasi lanjut termasuk Child Process dan Unit Testing.

## Cara Menjalankan Tes
Untuk memastikan semua fungsi berjalan dengan baik, jalankan perintah:
```bash
npm test
```
Proyek ini telah lulus pengujian kriteria submission Dicoding.
